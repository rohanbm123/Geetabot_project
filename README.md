# GeetaBot Project

GeetaBot is an AI-powered conversational assistant inspired by the Bhagavad Geeta. The project explores how LLMs and NLP can make ethical, reflective, and philosophical guidance more accessible through a simple conversational interface.

## Overview

The goal of this project is to create an assistant that can respond to user questions with context-aware guidance inspired by themes such as duty, discipline, purpose, emotional balance, and decision-making.

## Key Features

- Conversational interface for reflective and ethical questions
- Bhagavad Geeta-inspired response design
- Theme-based knowledge structure
- Simple explanations for complex philosophical ideas
- LLM/NLP-based conversational experience

## Tech Stack

- Python
- Google Colab
- NLP / LLM concepts
- Transformer-based language model workflow
